Navigation

Nehuba viewer contains with 4 main containers. 3 of them are 2D images of brain and each of them represents different dimentions of 3D image. In the forth container we can see 3D image of brain. With that page contains some panels for manage application and get information.
Let's Presend the simple page of atlas viewer

As we see, simple page contains with 8 basic parts. In the image most of parts are marked with square.
  1. 2D image of brain. From this panel user can change navigation and rotation.
  2. Same 2D image from another dimention.
  3. Same 2D image from another dimention.
  4. 3D image of brain. From this panel user can change navigation and rotation only for 3d image. User can not change position of camera.
  5. Panel contains with logo and icons for open floating widgets as searches, layers, plugins, etc.
  6. Panel contains information about position of cursor and camera. From camera it available to reset position, rotation and zoom for both 2D and 3D images.
  7. Panel contains information about zoom of in 2D images.
  8. From panel it is available to select template and for this template select parselation. Also it's available to search and select or deselect regions of brain. With what panel contains help and log in buttons.
